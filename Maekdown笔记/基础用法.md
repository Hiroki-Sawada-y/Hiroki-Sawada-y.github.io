## 段落
**段落的换行是使用两个以上空格加上回车**
```md
第一段+空格+空格+回车  

第二段
```
第一段  

第二段

### 字体
```md
*斜体*
_斜体_

**粗体**
__粗体__

***粗斜体***
___粗斜体___
```

### 分隔线
```md
***

* * *

*****

- - -

----------
```

### 删除线
```md
~~删除~~
```

### 下划线
```md
<u>带下划线文本</u>
```

### 脚注
```md

```
